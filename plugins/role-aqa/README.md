# Role-AQA: Senior AQA Engineer

Senior AQA Engineer persona with test strategy, E2E automation, API testing, performance testing, security testing, mobile testing, and CI integration expertise.

## Skills (8)

| Skill | Description |
|-------|-------------|
| `test-strategy` | Test pyramid design, risk-based prioritization, coverage goals, regression strategy |
| `e2e-testing` | Playwright, Cypress, Selenium, Page Object Model, visual regression, parallel execution |
| `api-testing` | SuperTest, REST Assured, contract testing (Pact), schema validation, mock servers |
| `performance-testing` | k6, JMeter, Gatling, Artillery, load/stress/soak testing, CI performance gates |
| `security-testing` | OWASP ZAP, SAST/DAST, dependency scanning (Snyk), penetration test planning |
| `mobile-testing` | Appium, Detox, XCUITest, Espresso, device farms, gesture and deep link testing |
| `test-data-management` | Factory patterns, database seeding, data isolation, anonymization, synthetic data |
| `ci-test-integration` | Parallel execution, test sharding, flaky test quarantine, Allure reporting, quality gates |
