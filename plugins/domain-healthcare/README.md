# domain-healthcare

Healthcare domain specialization plugin for Claude Code providing HIPAA compliance, HL7 FHIR interoperability, PHI data handling, clinical data modeling, EHR system integration, medical device software (IEC 62304), telehealth architecture, and healthcare analytics.

## Agents

- **healthcare-architect** - Health IT system architecture with HIPAA compliance by design, FHIR interoperability, clinical data standards, and medical device software guidance.
- **healthcare-compliance-advisor** - HIPAA regulatory compliance including Privacy Rule, Security Rule, BAA requirements, breach notification, and risk assessment.

## Skills

| Skill | Description |
|-------|-------------|
| `hipaa-compliance` | Technical, administrative, and physical safeguards; BAA checklists; risk assessment; breach notification |
| `hl7-fhir` | FHIR resources, RESTful API, SMART on FHIR, CDS Hooks, Bulk Data Access, US Core profiles |
| `phi-data-handling` | 18 HIPAA identifiers, de-identification (Safe Harbor, Expert Determination), access controls, audit logging |
| `clinical-data-modeling` | ICD-10, SNOMED CT, LOINC, RxNorm, CPT, NDC; CDA/C-CDA; patient data normalization |
| `ehr-integration` | Epic, Cerner/Oracle Health, Allscripts; HL7v2 messaging; Mirth Connect; CCD/C-CDA exchange |
| `medical-device-software` | IEC 62304 lifecycle, ISO 14971 risk management, design controls, FDA submissions, cybersecurity |
| `telehealth-architecture` | HIPAA-compliant video, remote patient monitoring, patient portals, store-and-forward, billing codes |
| `healthcare-analytics` | Population health, CDS, quality measures (eCQM, HEDIS, MIPS), OMOP CDM, SDOH data |
