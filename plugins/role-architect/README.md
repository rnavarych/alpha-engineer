# role-architect

Senior Architect plugin for Claude Code with 10+ years of system design experience.

## Skills

| Skill | Description |
|-------|-------------|
| **system-design** | Requirements analysis, C4 model diagrams, sequence diagrams, data flow diagrams, capacity estimation |
| **adr-writer** | Architecture Decision Records with structured format, numbering, status lifecycle, and team review |
| **scalability-analysis** | Horizontal/vertical scaling, sharding strategies, read/write splitting, CQRS, back-of-envelope calculations |
| **tech-stack-advisor** | Comparison matrices, TCO analysis, team capability assessment, ecosystem maturity, build vs buy |
| **migration-planner** | Strangler fig pattern, parallel run, data migration (ETL/CDC/dual-write), zero-downtime techniques |
| **cost-modeling** | Infrastructure cost estimation, TCO, ROI projection, pricing model comparison, breakeven analysis |
| **threat-modeling** | STRIDE methodology, attack trees, trust boundaries, risk assessment, mitigation prioritization |
| **capacity-planning** | Traffic estimation (DAU to QPS), resource sizing, growth modeling, peak load planning, load testing |

## Agent

- **senior-architect** — Full architect persona that combines all skills with cross-cutting concerns, domain adaptation, and architectural standards.
