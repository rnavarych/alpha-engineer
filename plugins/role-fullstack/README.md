# role-fullstack

Senior Fullstack Engineer plugin for Claude Code. Combines frontend and backend expertise for end-to-end development: scaffolding, API-frontend integration, auth flows, real-time features, and deployment workflows.

## Agent

- **senior-fullstack-engineer** -- Acts as a Senior Fullstack Engineer with 8+ years of experience. Thinks in vertical slices from database schema to API to UI, with emphasis on type safety across the stack, developer experience, and deployment simplicity.

## Skills

| Skill                      | Description                                                              |
|----------------------------|--------------------------------------------------------------------------|
| fullstack-scaffolding      | Scaffold production-ready projects with Next.js, Remix, Nuxt 3, SvelteKit |
| api-frontend-integration   | Type-safe API integration with tRPC, OpenAPI codegen, TanStack Query     |
| auth-flow                  | Login, signup, JWT, social login, MFA, protected routes, RBAC            |
| real-time-features         | WebSockets, SSE, presence, live cursors, collaborative editing, chat     |
| file-upload-handling       | Presigned URLs, image processing, drag-and-drop, validation, CDN serving |
| search-implementation      | ElasticSearch, Typesense, Meilisearch, faceted search, autocomplete      |
| deployment-workflow        | Vercel, Docker, preview environments, migrations in CI, rollback         |
| monorepo-management        | Turborepo, Nx, pnpm workspaces, shared packages, CI optimization        |
