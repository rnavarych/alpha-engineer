# role-devops

Senior DevOps Engineer plugin for Claude Code providing infrastructure, CI/CD, and operations expertise.

## Agent

- **senior-devops-engineer** - Acts as a Senior DevOps Engineer with 8+ years of experience in infrastructure automation, container orchestration, CI/CD pipelines, monitoring, incident response, cost optimization, and secrets management.

## Skills

| Skill | Description |
|-------|-------------|
| **docker-expert** | Multi-stage builds, image optimization, Docker Compose, networking, security scanning, registry management |
| **kubernetes-expert** | Deployments, services, ingress, HPA/VPA, RBAC, Helm, Kustomize, managed clusters (EKS/GKE/AKS) |
| **terraform-expert** | HCL modules, state management, workspaces, Terragrunt, infrastructure testing, drift detection |
| **pipeline-designer** | GitHub Actions, GitLab CI, Jenkins, ArgoCD, deployment strategies, quality gates, environment promotion |
| **monitoring-setup** | Prometheus/Grafana, Datadog, CloudWatch, USE/RED dashboards, alert routing, log aggregation |
| **incident-response** | Runbooks, postmortems, status pages, on-call rotation, escalation policies, chaos engineering |
| **cost-optimization** | Right-sizing, reserved/savings plans, spot instances, tagging, FinOps, storage lifecycle |
| **secrets-management** | Vault, AWS/GCP Secret Manager, SOPS, sealed secrets, rotation policies, zero-trust injection |
