# Testing Frameworks Reference

## JavaScript/TypeScript
- **Jest**: Unit/integration, built-in mocking, snapshot testing, code coverage
- **Vitest**: Vite-native, Jest-compatible API, faster for Vite projects
- **Playwright**: E2E, cross-browser, auto-wait, trace viewer, codegen
- **Cypress**: E2E, time-travel debugging, network stubbing, component testing
- **Testing Library**: DOM testing utilities, user-centric queries, framework adapters

## Python
- **pytest**: Fixtures, parametrize, plugins (pytest-cov, pytest-mock, pytest-asyncio)
- **unittest**: Built-in, class-based, setUp/tearDown
- **Hypothesis**: Property-based testing, automatic edge case generation

## Java/Kotlin
- **JUnit 5**: Annotations, parameterized tests, extensions, nested tests
- **Mockito**: Mock/stub/verify, argument captors, BDD API
- **TestContainers**: Docker containers for integration tests

## Go
- **testing**: Built-in package, table-driven tests, benchmarks
- **testify**: Assertions, mocking, suite
- **gomock**: Interface-based mocking

## API Testing
- **Postman/Newman**: Collections, environments, CLI runner
- **REST Assured**: Java DSL for REST API testing
- **SuperTest**: Node.js HTTP assertion library
- **Pact**: Consumer-driven contract testing
- **WireMock/MSW**: HTTP mock servers

## Performance Testing
- **k6**: JavaScript scripting, cloud execution, Grafana integration
- **JMeter**: GUI-based, protocol support, distributed testing
- **Gatling**: Scala DSL, detailed reports, CI integration
- **Artillery**: YAML config, WebSocket/HTTP support, easy CI integration

## Mobile Testing
- **Appium**: Cross-platform, WebDriver protocol
- **Detox**: React Native, gray-box testing
- **XCUITest**: iOS native, Xcode integration
- **Espresso**: Android native, synchronization
