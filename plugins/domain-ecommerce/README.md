# domain-ecommerce

E-Commerce domain specialization plugin for Claude Code. Provides payment gateway integration, product catalog design, cart/checkout flows, inventory management, order processing, search/merchandising, PCI DSS compliance, and analytics expertise.

## Agents

| Agent | Description |
|-------|-------------|
| `ecommerce-architect` | E-commerce platform architecture: catalog, cart, payments, orders, inventory, search, analytics |
| `ecommerce-payments-advisor` | Payment gateway integration, PCI compliance, subscription billing, fraud prevention |

## Skills

| Skill | Description |
|-------|-------------|
| `payment-integration` | Stripe, PayPal, Adyen integration; subscriptions; refunds; multi-currency; idempotency |
| `product-catalog` | Data modeling; pricing engine; tax calculation; localization; search indexing; images |
| `cart-checkout` | Cart persistence; checkout flows; guest checkout; shipping; promo codes; abandoned cart recovery |
| `inventory-management` | Stock tracking; warehouses; multi-location; backorders; reservations; stock alerts; channel sync |
| `order-processing` | Order state machine; fulfillment; returns/RMA; invoices; event sourcing; split shipments; notifications |
| `search-merchandising` | Search relevance; faceted navigation; recommendations; A/B testing; personalization; SEO |
| `pci-compliance` | SAQ levels; tokenization; secure payment forms; network segmentation; vulnerability scanning; audits |
| `ecommerce-analytics` | Conversion funnels; revenue attribution; cohort analysis; CLV; RFM; A/B test analysis; key metrics |
